Uni Timetable Exporter

My university's enrollment service did not offer any timetable export function.

This meant that students were forced to either take a screenshot of their class schedule or sign in to the enrollment service to check their class times.

A friend and I set about rectifying this by creating Uon Myhub Calendar Exporter a service that allows students to extract their class sechedule from the university enrollment service and add it to their calendar.

Raspberry Pi Robot

I made a robot! Using the chassy of an old remote control car, some servos, and a Raspberry Pi I was able to build an autonomous vehicle that can find it's way around obsticles.

Using an ultrasonic rangefinder, the robot is able to determine where obstacles are positioned and how far it is from them. It then tries to drive toward where there are the fewest obsticles.

I had loads of fun building the robot and experimenting with different behaviours
